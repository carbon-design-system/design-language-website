journeys.

## Component library

The component library provides developers a collection of reusable components to build pages that support IBM.com journeys. Adopting the library enables developers to use consistent markup, styles, and behavior in prototype and production work.

## IBM Digital Design kit

The IBM Digital Design kit contains core visual styles, components, color swatches, iconography, and pictograms. It also has a collection of symbols and text styles for rapid design and layout prototyping.
